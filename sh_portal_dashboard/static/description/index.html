Portal Dashboard

A dashboard is a view of geographic information that helps you monitor sale, purchase, invoice, project, lead, bills activities. Dashboards are designed to display multiple visualizations that work together on a single screen. Do you want to show a dashboard to your portal users? Do you want to show a well-formatted dashboard to your customer/vendor on my account screen? Do you want to show information on the lead, opportunity, quotation, sales, RFQ, purchase, invoices, bills, task, etc in graphical view? So here it is. we have made a materialize dashboard design for my account page. currently in odoo 'my account' have not well-formatted dashboard and is not well designed. we have modified and made that very beautiful and attractive.

Configuration Settings

Config Setting for Customer:

Sales Analysis (Sales Chart)   You can see the Sales analysis of the year chart in the dashboard.

Show Recent Quotation Table   You can see the recent quotation records in the table view.

Is Show Recent Sale Order Table   You can see the recent sale order records in the table view.

Invoice Analysis (Invoice Chart)   You can see the Invoice analysis of the year chart in the dashboard.

Show Recent Invoice Table   You can see the recent invoice records in the table view.

Config Setting for Vendor:

Bill Analysis (Bill Chart)   You can see the Bill analysis of the year chart in thedashboard.

Purchase Analysis (Purchase Chart)   You can see the Purchase analysis of the year chart in the dashboard.

Is Show Recent Purchase Order Table   You can see the recent purchase order records in the table view.

Show Recent RFQ Table   You can see the recent RFQ records in the table view.

Show Recent Bill Table   You can see the recent bill records in the table view.

Config Setting for Project and Task:

Is Show Recent Project Task Table   You can see the recent project and task records in the table view.

In config setting 'How many recent records do you want to show' set number of recent records which is shown in the dashboard.
